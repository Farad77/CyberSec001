3 vérifs avant de cliquer

  • Contexte : demande attendue, logique, datée, personnalisée.
  • Expéditeur : connu, adresse crédible (pas d’homographes).
  • Cible : lire le domaine (avant .fr/.com), pas de double extension (.pdf.exe).
Favoris officiels Lire le domaine Voir l’extension

Liens & URL

  • Domaine : login.orange.fr.attacker.com → réel : attacker.com.
  • Cadenas : chiffrement ≠ légitimité. Toujours lire le domaine.
  • Accès : banque/M365/Google via favoris ou appli officielle.

Pièces jointes

  • Danger par défaut : .exe, .msi, .iso, .zip et doubles extensions (.pdf.exe, .zip.exe).
  • Scanners/Transport : « scan.zip.exe », « étiquette.zip » → valider via un canal connu.
  • Macros : ne jamais activer ; préférer un viewer en ligne.

Mots de passe & MFA

  • Gestionnaire : utiliser celui du navigateur ; mots de passe uniques.
  • MFA : activer partout (mail, banque, cloud).
  • Fatigue MFA : refuser toute demande non sollicitée.

Téléphone & SMS

  • Vérifier via un numéro connu (site officiel/carte), jamais celui du message.
  • SMS livraison/banque : ouvrir l’appli officielle, pas le lien.
  • Ne jamais donner de codes MFA/banque par téléphone/SMS.

Web & téléchargements

  • Mises à jour/logiciels : uniquement depuis les sites officiels.
  • Liens raccourcis (bit.ly…) : prudence ; préférer l’accès direct au site/appli.
  • Navigateur à jour + protections (Safe Browsing/SmartScreen) activées.

Postes & réseaux

  • USB inconnue : ne pas brancher.
  • Wi‑Fi public : éviter pour banque/gestion ; sinon attendre ou VPN perso.
  • Verrouiller l’écran : Win+L dès qu’on s’absente.

En cas de doute

  • Ne pas cliquer/ouvrir ; ranger dans un dossier « À vérifier ».
  • Demander un 2ᵉ avis (collègue/dirigeant).
  • Contacter l’émetteur via un canal déjà connu (pas le numéro/lien du message).
Astuce : une pause de 30 secondes aide à repérer les incohérences et les urgences artificielles.

En cas de clic / téléchargement

  • Couper Internet (Wi‑Fi/ethernet). Ne pas exécuter le fichier.
  • Noter l’heure/le message et faire une capture.
  • Scanner avec Windows Sécurité ; ne pas « nettoyer » seul si alerte.
  • Changer les mots de passe sensibles depuis un appareil sain.
  • Paiements : appeler la banque/fournisseur via numéro officiel pour bloquer/valider.

Sauvegardes simples

  • Hebdo : copie des dossiers clés sur disque externe (débranché hors usage).
  • Cloud pro minimal (OneDrive/Drive) pour documents critiques.
  • Tester une restauration 1×/trimestre.

Réglages express Windows (5 min)

  • Extensions visibles : Explorateur → Affichage → cocher « Extensions de noms de fichiers ».
  • SmartScreen/Réputation : Sécurité Windows → Contrôle des applis & du navigateur → activer.
  • Mises à jour auto : Windows Update + Navigateur (Chrome/Edge/Firefox).

Procédure maison (à afficher)

  • Référent : [Nom] — tél : [Numéro].
  • Virements/achats urgents : jamais seul ; double validation systématique.
  • Fiche d’incident : date, canal, action (utile assurance/suivi).
